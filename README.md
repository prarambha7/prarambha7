## 🛠 Setup Notes (to activate "live" elements)

1. **Replace placeholders** — swap `prarambha` in the stats/snake URLs and the `YOUR-LINKEDIN` / `YOUR-EMAIL` links with your real details.
2. **GitHub Stats / Streak / Activity Graph** — these are hosted, dynamically-generated images (no setup needed beyond your username being correct). They refresh automatically every time someone views your profile.
3. **Contribution Snake animation** — this one *does* need a small one-time setup:
   - Create a file at `.github/workflows/snake.yml` in a repo named exactly like your username (your GitHub profile repo).
   - Use the [`Platane/snk`](https://github.com/Platane/snk) GitHub Action (search "github contribution snake action" for the copy-paste workflow).
   - It runs on a schedule, generates an SVG, and commits it to an `output` branch — which is what the image above points to.
4. **Profile views counter** — the CountAPI badge works out of the box, but CountAPI has had reliability issues; a common stable alternative is [`komarev.com/ghpvc`](https://komarev.com/ghpvc/) — happy to swap it in if you'd like.

If you want, I can also make a **dark/light adaptive version** (stats that switch theme based on GitHub's color mode) or a version with a custom animated SVG banner instead of capsule-render.
